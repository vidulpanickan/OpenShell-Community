Bring Your Own Agent

Connect from your laptop, create a sandbox, and run any agent inside it. The NemoClaw TUI on your machine surfaces policy recommendations from the agent. You approve or deny — the sandbox boundary is never violated.

1. Install NemoClaw CLI

pip install nemoclaw

2. Connect to this VM

nemoclaw cluster connect <loading...>

3. Create a sandbox and run your agent

# Claude Code nemoclaw sandbox create -- claude # OpenCode nemoclaw sandbox create -- opencode # Any agent nemoclaw sandbox create -- your-agent-command

4. Manage policies with the TUI

nemoclaw term

When your agent hits a policy denial, it reads the policy, diagnoses the block, and proposes an update. The TUI surfaces the recommendation: "Your sandbox wants to connect to pypi.org. Recommended policy update: [allow pypi.org:443]. Approve?" You approve — the policy hot-reloads — the agent continues.
